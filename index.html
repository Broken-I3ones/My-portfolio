My Work

HTML
HTML (Hyper Text Markup Language) Is standardized for making websites its what makes the whole structure of the website by using different elements. When it comes to structure html is the bones, If you wanted to make a decent website without html it would be extremely difficult.
CSS
CSS (Cascading Style Sheets) allows you to make a great looking website, with css you can change the color of backgrounds and texts, you can resize anything on the website, you can create boxed lists, ext. In a structural setup CSS is the clothes of the website to make it look flashy and nice.
Javascript
Javascript is a scripting language that allows complex features such as displaying content updates, interactive maps, animated 2d/3d graphics, ext. When it comes to the structure of the website its what gives the website the soul that it needs to function.
